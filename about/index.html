So I've been reading a lot of online blogs throught the years (Crackberry, Hackaday), and I've always wanted to have somewhere to share my thoughts and my projects.

Some of you might know I've got a youtube channel, but honestly I have a lot of trouble with finishing a video, let alone a decent one. So I'd like to focus on something I feel I can keep up with. Hopefully I can update here somewhat consistently, and pardon any spelling mistakes since I'm using a code editor rather than a word processor.

If you're interested in the source code of this blog it is available here. It is designed to be compatible with old browsers, specifically the BlackBerry 8520's. I do know that Jekyll has support for blogs, but doing it this way makes it easier for me to do custom cool stuff.
